# Registration Form

A modern and responsive registration form built using **HTML** and **CSS**. This project focuses on creating a clean, user-friendly interface with a visually appealing layout and responsive design.

## ✨ Features

- Responsive design
- Clean and modern UI
- User-friendly layout
- Mobile-friendly interface
- Easy to customize

## 🛠️ Technologies Used

- HTML5
- CSS3

## 📂 Project Structure

```
Registration-Form/
│── index.html
│── style.css
└── README.md
```

## 🚀 Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/Registration-Form.git
   ```

2. Open the project folder.

3. Open `index.html` in your preferred web browser.
